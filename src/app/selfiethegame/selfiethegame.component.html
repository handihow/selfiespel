Introduction Short introduction to SelfieTheGame Video on YouTube Create game How to create a new game in SelfieTheGame Video on YouTube
Step 1 Start a new game as an administrator

You can start a new game by logging into this website. Using the QR code (scanning) or the code (entering), players can log into your game.

You can then set the commands in the form of "Take a selfie with ...". After that you can possibly add jury members. The administrator is automatically a jury member and can also give points. Finally, you put the teams together. This is only possible after everyone has registered.

The easiest way to manage a new game is to log in to this website.

 LOG IN Step 2 Teams take selfies with the assignment

Once the game has started, the teams can take selfies with the assignments. Everyone can see the submitted selfies of all teams.

You can like selfies and there is a chat room where you can communicate with each other.

The administrator can pause and restart the game in the meantime. When the game is paused or finished, players cannot submit selfies.

It is best for players to use the apps in the Google Play or iOS App Stores.

 GOOGLE PLAY STORE IOS APP STORE Step 3 The administrator (and jury members) assess the selfies

Judges (including the administrator) can rate the selfies during or after the game.

You have already set the maximum number of points per selfie in the settings of the game. You can never give more points than the maximum number of points for the assignment, but you can give less.

The points are added together and the team with the most points wins the game!
